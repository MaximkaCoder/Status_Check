# Що додати

1. Ролі користувачів: ADMIN, MANAGER, MEMBER, VIEWER
2. Відділи — модель Department, прив'язка юзерів до відділів
3. Членство в проєктах — хто має доступ, яка роль
4. Адмін-панель: управління юзерами, відділами, проєктами
5. Коментарі до задач
6. Аудит-лог — хто що змінив і коли
7. Email-сповіщення: призначено задачу, підходить дедлайн, змінено статус
8. Пошук по задачах (title, description)
9. Розширені фільтри: по виконавцю, проєкту, діапазону дат, пріоритету
10. Поле пріоритету (HIGH / MEDIUM / LOW / CRITICAL)
11. Архів — видалені задачі не зникають назавжди
12. Аналітика: графіки виконання, навантаження по команді, експорт у CSV
13. Інтеграції: Slack / Telegram сповіщення
